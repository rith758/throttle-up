📌 Project Overview

Throttle Up is a modern web application designed for bikers to connect, organize, and participate in events such as rides, meetups, and campaigns.

The platform allows users to act as both organizers and participants, helping build a strong biking community through shared interests and activities.

---

🚀 Features

👤 User System

- User registration and login
- Personal profile with biking interests
- Track events created and joined

---

📅 Event Management

- Organizers can create events with:
  - Title
  - Description
  - Image/Media
  - Location
  - Date & Time
- Events are stored and dynamically displayed

---

🔍 Event Discovery

- Browse all available events
- Responsive event cards UI
- Join events of interest

---

🌐 Community Building

- Connect with bikers of similar interests
- Participate in rides, campaigns, and meetups
- Build a biking network

---

🛠️ Tech Stack

Frontend

- React.js (Vite)
- Tailwind CSS

Backend & Database

- Supabase (Authentication + Database)

Deployment

- Vercel

---

⚙️ How It Works

- The frontend is built using React and communicates with Supabase APIs
- Supabase handles:
  - User authentication
  - Event data storage
- The application is deployed on Vercel for public access

---

📷 UI Highlights

- Modern dark-themed UI
- Responsive design (mobile + desktop)
- Interactive event cards
- Smooth navigation

---


📚 Conclusion

Throttle Up provides a user-friendly platform for bikers to explore events, connect with others, and actively participate in the biking community. It combines modern UI design with real-time backend services to deliver a seamless experience.
